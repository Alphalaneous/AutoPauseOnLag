# Auto Pause on Lag

Automatically pauses the level you are playing if the game lags.

Adjust the length of time the lag spike must be in the mod's settings.

<c-ff765e>This mod is considered a </c><c-6e6e6e>grey area</c><c-ff765e> on if it is cheating or not, use at your own risk. Don't abuse this to cheat, there are measures to prevent pause buffering using this in place already, but we can never know what someone clever figures out.</c>
